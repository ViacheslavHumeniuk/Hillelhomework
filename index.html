Terminal Command

  • git init: Ініціалізує новий репозиторій Git в поточному каталозі.
  • git clone URL: Клонує віддалений репозиторій Git на вашому локальному комп'ютері.
  • git add файли: Додає зміни в підготовлену область для коміту. Ви можете вказати конкретні файли або використовувати . для додавання всіх змін.
  • git commit -m "Ваш коментар": Комітує зміни, які знаходяться в підготовленій області, з обов'язковим коментарем.
  • git status: Виводить інформацію про поточний стан вашого робочого каталогу та підготовленої області.
  • git log: Показує історію комітів.
  • git branch: Показує список гілок в репозиторії та вказує поточну гілку.
  • git checkout гілка: Перемикається на іншу гілку.
  • git pull: Витягує зміни з віддаленого репозиторію та об'єднує їх з вашою локальною копією.
  • git push: Відправляє ваші локальні коміти на віддалений репозиторій.
  • git merge гілка: Об'єднує обрану гілку з поточною гілкою.
  • git pull --rebase: Витягує зміни з віддаленого репозиторію та перезапускає ваші локальні коміти поверх них.
  • git reset файли: Скасовує зміни в підготовленій області для вказаних файлів.
  • git stash: Використовується для відкладання змін в тимчасовий сховище.
  • git remote -v: Виводить список віддалених репозиторіїв та їх URL-адреси.
